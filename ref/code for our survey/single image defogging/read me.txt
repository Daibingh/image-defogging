demo_DCP_soft_matting.m: the dark channel priori based defogging algorithm by using soft matting method to                          optimize the transmission.

demo_DCP_filtering.m   :the dark channel priori based defogging algorithm by using bilateral filtering, joint                         bilateral filtering and guided filtering for transmission potimization.

demo_CLAHE_defogging.m : using CLAHE to enhance the foggy image.
